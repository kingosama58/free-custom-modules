KSA Custom POS Receipt

The KSA Custom POS Receipt module provides a fully customizable Point of Sale (POS) receipt template compliant with Saudi Arabian regulations. It includes VAT details and other necessary information for legal and business purposes.


Features

  • Customizable POS receipt layout
  • Includes VAT details as per KSA regulations
  • Supports additional company information (logo, phone, email, website)
  • Seamless integration with the Odoo Point of Sale module

Pricing

Price: $30 USD

Dependencies

  • Odoo Base Module (base)
  • Odoo Point of Sale Module (point_of_sale)
